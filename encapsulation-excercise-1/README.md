# Encapsulation Practice 01

## Goal
Practice basic encapsulation in Java using private fields and getters/setters.

## Task

Implement a `Person` class that stores a person's name.

### Requirements

- The `name` field must be **private**
- Implement the following methods:


- `setName` should store the name
- `getName` should return the stored name

---

## Important

❗ Do NOT modify the tests.

Your goal is to implement the class so that all tests pass.

---

## How to Run

1. Open the project in IntelliJ
2. Reload the Maven project
3. Run the tests inside `PersonTest`
4. Implement `Person.java` until all tests pass

---

## Learning Objective

Understand:

- private fields
- getters
- setters
- encapsulation basics
