# Encapsulation Practice 03

## Goal

Practice encapsulation with multiple fields.

## Task

Implement a `Person` class that stores basic profile information.

## Requirements

The class must contain the following private fields:

String name  
int age  
double height

Implement getters and setters for all fields.
setName(String name)
getName()

setAge(int age)
getAge()

setHeight(double height)
getHeight()


### Validation Rules

Age:
- must be between 0 and 130

Height:
- must be greater than 0
- must be less than 300

If invalid values are provided, the value should **not change**.

## Important

Do not modify the tests.

Make your implementation pass all tests.
